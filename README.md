# 真武汤·附子奇缘

这是一个关于中医药文化的网页互动游戏。玩家将扮演一名中医学徒，在师父的指引下，通过七个关卡的考验，深入了解“真武汤”及其君药“附子”的性味、功效、炮制、配伍及解毒方法。

## 游戏内容

游戏共包含七个关卡：
1.  **炮制之辨**：辨别生附子与熟附子。
2.  **生熟之异**：理解生熟附子的功效差异。
3.  **剂量之衡**：掌握附子的安全用量。
4.  **煎煮之秘**：学习附子的正确煎煮方法（久煎）。
5.  **毒象之辨**：识别附子中毒的症状。
6.  **解毒之策**：掌握附子中毒的急救与解毒药物。
7.  **配伍之妙**：理解真武汤中各味药材（特别是生姜）的配伍意义。

## 技术栈

*   HTML5
*   CSS3 (包含动画效果)
*   JavaScript (原生)

## 如何运行

### 本地运行
直接在浏览器中打开根目录下的 `index.html` 文件即可。

### 在线预览
本项目支持部署到 GitHub Pages。

## 目录结构

*   `public/`: 包含所有关卡的 HTML/CSS 源码。
*   `assets/`: 存放图片、音频等静态资源。
*   `index.html`: 入口文件，负责重定向到开始页面。
